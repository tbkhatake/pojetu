Lição básica de tamanho

No CSS temos certos códigos para alterar o tamanho dos objetos:

width(altera largura)
height(altera altura)
padding(altera o espaçamento interno), mas mexe no tamanho total também.
margin(altera o espaçamento interno)

  • img src = "images/helloworld.png" - adiciona imagem
  • link rel="stylesheet" href="css/styles.css" - adiciona o arquivo com o css no html
  • display:inline,block ou inline-block - Muda a forma de como o objeto ocupa o espaço na página
  • vertical-align:top,bottom - muda o alinhamento do objeto na linha em que ele ocupa
  • position relative - Posição relativa ao ponto inicial do item
  • position absolute - Posição relativa a pagina ou dentro de um item com o position relative
